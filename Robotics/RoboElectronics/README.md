# RoboElectronics
Extracurricular, student-run semestrial robotics project  

__TASK__ Build and program a robot that can go on a seesaw and balance itself on it   


__COMPONENTS__
-  Breadboard
-  Pyboard
-  Heat Disipator
-  H-Bridge
-  Accelerometer
-  Infrared Sensor
-  DC Motors
